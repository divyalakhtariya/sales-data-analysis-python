# Sales Analysis Using Python

## Project Overview
This is my first Data Analyst project.
In this project, I analyzed sales data using Python to answer business questions related to sales and profit.

---

## Questions Solved
1. Monthly sales analysis (highest and lowest month)
2. Sales analysis by product category
3. Sales analysis by sub-category
4. Monthly profit analysis
5. Profit analysis by category and sub-category
6. Sales and profit analysis by customer segment
7. Sales to profit ratio analysis

---

## Tools Used
- Python
- Pandas
- Plotly
- Jupyter Notebook

---

## Dataset
Superstore dataset
